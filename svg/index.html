        *:
            fill: 填充色
            stroke: 描边色
            stroke-width: 描边宽
        rect: 矩形
            x=0: 相对svg横坐标
            y=0: 相对svg纵坐标
            width: 宽
            height: 高
        circle: 圆
            cx: 圆心x
            cy: 圆心y
            r: 半径
        ellipse: 椭圆
            cx: 圆心x
            cy: 圆心y
            rx: x轴半径
            ry: y轴半径
        polyline: 折线，有默认背景色，通过 fill="transparent" 去掉
            points: 折点 ="x1 y1, x2 y2, ..."
        polygon: 多边形，自动闭合
            points: 折点 ="x1 y1, x2 y2, ..."
        path: 路径
            d: 命令="M x y  L x y  H x  V y  C x1 y1, x2 y2, x y  ...  z" ，大写绝对，小写相对

                L: lineTo
                H: 水平画
                V: 垂直画
